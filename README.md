# 🌟 Colorful Spiral Star

A colorful spiral star pattern created using Python's Turtle Graphics.

## Features

- Multiple color animation
- Black background
- Geometric spiral pattern
- Beginner-friendly Python project

## Technologies Used

- Python
- Turtle Graphics

## How to Run

1. Install Python.
2. Save the code as `colorful_spiral_star.py`.
3. Run the file.

## What I Learned

- Using loops
- Working with lists
- Changing colors dynamically
- Turtle Graphics
- Creating geometric patterns
